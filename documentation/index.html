WOOFER Project Description

Woofer is a full-stack dog-themed Twitter Clone. Users can create a new account and login. Users can create new posts (called woofs) and comment on other users woofs. Users can follow other users and like their woofs. Technologies Used

Angular
Bootstrap
Spring Boot
Postgres

Features

Users can register a new account.
Users can login to their account.
Users can create new posts(woofs) as well as edit, and delete their posts.
Users can view other users woofs.
Users can like posts and follow other users.
